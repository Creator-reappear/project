Types of formatting tags

  1. bold tag or strong tag
  2. italics tag or em tag
  3. mark tag or highlighting tag
  4. inserting tag or underline tag
  5. deleting tag or strike tag
  6. supercript tag
  7. subscript tag

Example for bold tag:-
I'm using phone in class
Example for italics tag:-
I went to Delhi
Example for mark tag:-
Take a look at page no 615
Example for inserting tag:-
You are Beautiful
Example for deleting tag:-
I killed saved many people
